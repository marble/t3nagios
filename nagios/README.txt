Nagios TYPO3 Monitoring (TYPO3 extension "nagios")

Description:
Nagios is the industry standard in open source server/service/network monitoring.
This TYPO3 extension gathers information about the TYPO3 instance for a Nagios monitoring server.

Read full documentation at: http://schams.net/nagios
